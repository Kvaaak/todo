# 📱 My Expo App

A simple multi-tool and mini-game app built with React Native and Expo.

The app includes useful tools like a calculator and to-do list, as well as fun mini-games.

---

## ✨ Features

### 🛠 Tools

* 🧮 Calculator (basic operations, square root, percentage)
* 📝 To-do list (add & manage tasks)

### 🎮 Games

* ✊ Rock Paper Scissors
* 👻 Random Ghost Generator

---

## 🚀 Tech Stack

* React Native
* Expo
* JavaScript
* React Hooks

---

## 📦 Getting Started

### 1. Install dependencies

npm install

### 2. Start the app

npx expo start

### 3. Run on device

* Scan QR code with Expo Go app
* Or run on emulator

---

## 📁 Project Structure

/app

* Application screens and navigation

/assets

* Images, icons, and other static files

/components

* Reusable UI components

/constants

* Theme and constant values

/context

* Global state management (React Context)

/hooks

* Custom React hooks

/lib

* Utility functions and helpers

/scripts

* Project scripts and automation

/types

* TypeScript type definitions
